## Ejemplos Apple Watch

En este proyecto se ven diferentes ejemplos para hacer aplicaciones en Apple Watch con **WatchOS 2**. Hay ejemplos de interfaz, comunicación entre iOS y watchOS, servicios y persistencia, creación de glances, complications y notificaciones, y utilización del hardware.

#### Ejercicios

A continuación, enumero los diferentes ejercicios con una breve explicación de cada uno:

- Ejercicio 1: Interfaces 1, con ejemplos de uso de los **elementos básicos de la interfaz** y siguiendo una navegación lineal
- Ejercicio 2: Interfaces 2, con un ejemplo del uso de **mapas** y la navegación con paginador
- Ejercicio 3: Interfaces 3, con un ejemplo del uso de **vídeos**, **menú contextual** y la mezcla de navegaciones
- Ejercicio 4: **Notificaciones**, en el que se muestra como recibirlas, diseñarlas, hacer pruebas en el simulador y añadir botones reales
- Ejercicio 5: **Glances**, donde se muestra como diseñar esta pantalla y mostrar los datos
- Ejercicio 6: **Complicaciones**, con el código para ver los datos desde la vista del reloj
- Ejercicio 7: **Hardware**, con ejemplos de uso del micrófono, sensores (como el acelerómetro o el pulsómetro), llamadas, etc.
- Ejercicio 8: Interfaces 4, con ejemplos extendidos del uso del **WKInterfacePicker**
- Ejercicio 9: **Watch Connectivity**, en las que se explica como utilizar cada uno de los métodos para comunicarse con iOS
- Ejercicio 10: Interfaces 5, con diferentes ejemplos sobre como utilizar **imágenes y animaciones**
- Ejercicio 11: **Servicios y persistencia**, donde se muestran ejemplos básicos de descarga de archivos y cómo guardarlos en el Apple Watch, así como su integración con **CocoaPods**
